ntopng
 docker run -it -p 3000:3000 -v $(pwd)/ntopng.license:/etc/ntopng.license:ro --net=host ntop/ntopng:latest -i enp1s0
Внимательно указываем свой интерйейс -i enp1s0 Трафик с указанного интерфейса enp1s0 сразу же начнет анализироваться.

После этого идём в веб интерфейс http://ip:3000/, учётка admin / admin.

Основные возможности ntopng:

Веб интерфейс для управления
Визуализация трафика в режиме реального времени
Фильтрация и группировка трафика по различным признакам
Группировка хостов по различным признакам (ГЕО, AS, система..)
Хранение исторических данных в БД
Формирование предупреждений на основе различных событий
Отправка оповещений по email, telegram, discord и т.д.
