Для отслеживания скорости записи данных на диск в Linux Debian вы можете использовать утилиту iostat. Iostat предоставляет информацию о вводе-выводе дисковой подсистемы, включая скорость записи данных.

apt-get install sysstat

Запустите iostat, указав интервал обновления и количество отображаемых строк. Например, для вывода обновлений каждые 1 секунду и отображения 5 строк выполните следующую команду:

iostat -d 1 5
