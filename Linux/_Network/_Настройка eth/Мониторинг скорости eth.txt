iftop -i eth0

2. Для отображения статистики по отдельным хостам используйте опцию "-F" (filter):

sudo iftop -i eth0 -F 192.168.0.1/24

3. Для вывода статистики в более удобном формате можно воспользоваться опцией "-B" (burst mode). 
Эта опция позволяет выводить статистику не непрерывно, а только при появлении трафика в сети:

sudo iftop -i eth0 -B

4. Для отображения суммарной статистики использования сетевого интерфейса воспользуйтесь опцией "-t" (totals):

iftop -i eth0 -t
