текущая нагрузку на диск
apt-get install iotop -y

iotop -o

По умолчанию, интервал мониторинга составляет 3 секунды. Чтобы изменить интервал мониторинга, можно использовать опцию -d, например
iotop -o -d 5


PID идентификатор процесса;
PRIO приоритет процесса;
USER имя пользователя, от имени которого запущен процесс;
DISK READ количество данных, прочитанных процессом с диска за последний интервал;
DISK WRITE количество данных, записанных процессом на диск за последний интервал;
SWAPIN количество данных, прочитанных процессом из области подкачки за последний интервал;
IO общее количество операций ввода-вывода (читать и записывать) за последний интервал;
COMMAND имя процесса.
