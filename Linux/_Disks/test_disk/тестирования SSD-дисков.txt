Запустите команду sudo smartctl -a /dev/sda 
для вывода полной информации о состоянии диска /dev/sda. 


Запустите тест S.M.A.R.T., чтобы проверить наличие проблем с диском. 
Вы можете запустить короткий тест, 
выполнив команду 
sudo smartctl -t short /dev/sda, 
или полный тест, выполнив команду 
sudo smartctl -t long /dev/sda. 

После запуска теста вы можете использовать команду 
sudo smartctl -a /dev/sda для проверки результата тестирования.
