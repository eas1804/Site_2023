В Windows можно включить функцию Wake-on-LAN (WOL) для сетевого интерфейса с помощью утилиты powercfg. 

1/ Откройте командную строку с правами администратора.

Введите следующую команду, чтобы просмотреть список доступных сетевых адаптеров:

powercfg /devicequery wake_armed

Вы увидите список сетевых адаптеров, которые уже настроены для активации по событию WOL.

Для включения WOL для определенного сетевого адаптера выполните команду:

powercfg /deviceenablewake "<Имя сетевого адаптера>"

Замените <Имя сетевого адаптера> на имя конкретного сетевого адаптера, для которого вы хотите включить WOL. 
Вы можете использовать имя, отображаемое в списке из предыдущего шага.

Убедитесь, что WOL успешно включен для сетевого адаптера, выполните команду:

powercfg /devicequery wake_armed

В списке должен появиться ваш сетевой адаптер с пометкой "Wake Enabled".

Теперь ваш сетевой адаптер в Windows настроен для активации по событию WOL. При получении правильного магического пакета ваш компьютер должен включиться через сетевой интерфейс, подключенный к этому адаптеру.
