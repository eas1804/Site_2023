Учетные записи

Просмотр списка пользователей
net user

Вывести информацию по текущему пользователю
net user USER_NAME

Установка пароля пользователя Администратор
net user Администратор NEW_PASSWORD

Создать пользователя
net user NEW_USER /add

Добавить пользователя в группу Администраторы
net localgroup Администраторы USER_NAME /add
также, может быть «administrators», вместо «Администраторы»

Удалить пользователя из группы
net localgroup Администраторы USER_NAME /delete

Как включить встроенную учетную запись администратора в Windows 7/10
net user Администратор /active:yes
/active:{yes | no}

Срок действия пароля — не заканчивается.
!!!Не забыть кавычки для имени пользователя
wmic path Win32_UserAccount where Name=»userName» set PasswordExpires=false

Запретить менять пользователю пароль
net user администратор /passwordchg:no

В какие группы входит пользователь
net user USER_NAME |findstr Членство

удаление учетной записи:
net user petr /delete;

Запуск от имени другого пользователя:
runas /user:administrator «C:\Windows\cmd.exe»
runas /savecred:/user:admin@mydomain «C:\Windows\cmd.exe» — с сохранением пароля

Скрыть пользователя с экрана привития Windows
reg add «HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList» /v Ира /t reg_dword /d 0
