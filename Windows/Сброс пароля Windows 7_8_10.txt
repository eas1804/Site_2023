Замена Utilman.exe на cmd.exe для запуска команды net user

1. Загрузитесь с установочного диска Windows
2. Shift+F10
3. copy D:\Windows\System32\Utilman.exe D:\
Резервная копия файла Utilman.exe
4. copy D:\Windows\System32\cmd.exe D:\Windows\System32\Utilman.exe
Подмена файла Utilman на cmd.exe
5. Перезагружаемся. 
На экране блокировки нажмите кнопку вызова специальных возможностей, 
и должно появиться окно командной строки. 
Таким образом, с помощью командной строки установочного диска мы запустили командную строку Вашей системы, в которой необходимо выполнить следующую команду:
6. net user Admin 123
После выполнения указанных выше действий следует вернуть  на место Utilman.exe. Для этого надо обратно загрузиться с установочного диска иmove D:\Utilman.exe D:\Windows\System32\Utilman.exe
